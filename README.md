Katlogic 

Katlogic is a modern AI-powered tools application that provides users with multiple smart tools in one place. The app is designed to improve productivity, creativity, and efficiency using the power of artificial intelligence.

 Features
	•	 AI Chat Assistant – Ask questions and get intelligent responses.
	•	 Multiple AI Tools – Access different AI utilities in one application.
	•	 Fast and Responsive UI – Smooth user experience built with Flutter.
	•	 Modern Design – Clean and simple interface.
	•	 Secure Usage – Safe interaction with AI tools.

 Built With
	•	Flutter – Cross-platform app development
	•	Dart – Programming language used by Flutter
	•	AI APIs – For intelligent responses and tools
